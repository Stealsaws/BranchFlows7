  4. Add bot เข้ากลุ่ม LINE ที่ต้องการแจ้งเตือน
  5. รับ Group ID จาก Webhook event (ใช้ webhook.site ช่วยได้)
  6. เพิ่ม action:'lineMessage' ใน Apps Script ของคุณ
